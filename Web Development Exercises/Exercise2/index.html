Λίγα πράγματα για εμένα

Είμαι 4ετής Φοιτήτρια στο Τμήμα Πληροφορικής του Οικονομικού Πανεπιστημίου Αθηνών. Οι κύκλοι μαθημάτων που επέλεξα είναι οι εξής:

  • Πληροφοριακά Συστήματα και Ασφάλεια
  • Βάσεις Δεδομένων και Διαχείριση Γνώσης

και ο λόγος που έκανα αυτές τις επιλογές είναι ότι το ενδιαφέρον μου εστιάζει κυρίως στις βάσεις δεδομένων και την ανάπτυξη λογισμικού. Ως γλώσσα προγραμματισμού χρησιμοποιώ κυρίως την γλώσσα Java, αλλά έχω ασχοληθεί και με τη γλώσσα Python για ανάλυση δεδομένων και βασικές εφαρμογές μηχανικής μάθησης. Επίσης έχω βασικές γνώσεις στις γλώσσες C/C++, χωρίς ωστόσο να έχω μεγάλη ευχέρεια σε αυτές όπως στην Java ή την Python.
